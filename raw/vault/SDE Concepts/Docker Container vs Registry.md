# Docker Container vs Registry

**Conversation Summary**

1.	**CI/CD Pipelines & Jenkins** – Discussed how CI/CD automates software development, testing, and deployment. Jenkins was introduced as a CI/CD tool, with an example **Jenkinsfile** for automation.

2.	**Docker Registry** – Explained as a storage system for Docker images, differentiating **public** (Docker Hub, AWS ECR) and **private** registries.

3.	**Docker Container vs. Docker Repository** – Clarified that a **Docker container** is a running instance of an image, while a **Docker repository** stores multiple versions of an image in a registry.

4.	**Docker Compose** – Introduced as a tool for managing multi-container applications using a docker-compose.yml file. Covered an example with **Flask + PostgreSQL**, key features, and essential commands.
